Static - Default positioning behavior. Element is positioned according to the normal flow of the document.
Relative - Element is positioned relative to its normal position in the document flow.
Absolute - Element is removed from the normal document flow and positioned relative to its closest positioned ancestor.
Fixed - Element is removed from the normal document flow and remains fixed relative to the viewport.
Sticky - Element toggles between relative and fixed positioning based on the user's scroll position.
